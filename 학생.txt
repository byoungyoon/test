학생 요구 별 페이지

index(초기)
	- 달력출력
	- 수강중인 강의 목록
	- 공지 사항
	- 켜뮤니티게(게시판)

마이페이지
	- 개인정보 수정기능(사진 포함)
	- 자신의 파일올릴 공간 존재
	- 회원 탈퇴
	- 평균 성적, 학기별 성적을 보여주는 통계
	- 내 수강 내역 조회
	- 내 개별 시간표
	- 결제 강의, 결제, 결제날짜 확인

강의 페이지
	- 나와 관련있는 강의 리스트 출력(이수, 이수중, 신청중) 상태 확인
	- 나와 관련 없는 강의 리스트 전체 출력
	- 그와 관련된 장바구니 기능
	- 수강신청 목록 추가
	- 강의별, 강사별 검색하여 평가내역 확인

강의 상세보기 페이지
	- 수업 영상 및 시간 체크
	- 수업 정보, 강사, 기간, 진행률 표시

나의 강의 페이지
	- 성적 확인 기능
	- 수업 평가
	- 수업 진행확인
	- 달력에서 현재 출결 여부를 일별로 보여주고 시험과 같은 일정을 보여주는 기능
	- 과제가 나오면 제출할 수 있도록 첨부파일을 추가할 수 있는 기능
	- 수업 캘린더( 수업 날짜에 개인 일지 작성 가능)
	- 출석체크

강의에 따른 qna 페이지
	- 강사님과 관리자만 볼 수 있는 상담 작성, 질의 응답
	- 강사와의 1:1쪽지 기능
	- 수신, 발신리스트 작성

후기 페이지
	- 강의 및 강사 평가 참여
	- 수료 후기 작성
