## 마케팅 자동화 프로그램
> 네이버 크롤링과 chartGPT를 이용하여 마케팅을 좀 더 효율적으로 하기 위한 도구

### 개발  환경

- Next.js (13.x), css, vanilla extract, zustand, 
```bash
run: npm run dev
build: num run build
```

### 프로젝트 구조

> App Router

```bash
├── public
│   └── favicon, fonts, images, svg ...
├── app
│   ├── @panel
│   │   │   └── (.)i
│   │   │       └── [project]
│   │   │           ├── readme
│   │   │           └── skill
│   │   ├── _component
│   │   └── i
│   │       ├── readme
│   │       └── skill
│   └── _data
└── next.config.js, .prettierrcs.js, eslintrc.json ...
```
